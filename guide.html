API Usage Guide

This guide provides examples of how to use different models in our API. Please replace <your_api_key> and <your_question> with your actual API key and question. For the YouTube model, replace <youtube_video_url> with the actual URL of the YouTube video. For the Amazon model, you can also specify the number of products with the product parameter.

1. Amazon Model

import requests
import json

url = "http://panel.magicgames.mooo.com:1037/api"
data = {
    "api_key": "<your_api_key>",
    "model": "amazon",
    "question": "<your_question>",
    "product": 10
}

response = requests.post(url, data=json.dumps(data))
print(response.json())
